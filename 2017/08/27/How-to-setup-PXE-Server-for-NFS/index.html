3. 编译客户端启动内核

编译客户端启动所用的内核,要注意的地方:

  • 网络在内核启动过程中的支持
  • 块设备在内核中的支持
  • 支持远程 PXE 启动的网卡驱动不要作为独立的模块编译
  • 网络文件系统的支持

以上提到的只是最基本的支持,需要根据自己的需要对内核做实际的配置。
内核源代码可以从网页 http://www.kernel.org/pub/linux/kernel/ 下载。
